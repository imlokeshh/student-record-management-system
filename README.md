# Student Record Management System

A Java console application to manage student records using OOP and Collections.

## Features
- Add a student
- Display all students
- Delete a student by ID
- Search a student by ID

## Tech Stack
- Java
- ArrayList (Collections)
- OOP principles (Classes, Constructors, Methods)

## How to Run
1. Clone this repo
2. Open in any Java IDE or Programiz
3. Run `StudentManager.java`

## Sample Output
Student added: Lokesh
Student found: Priya (ID: 3)
Deleted: Rahul
